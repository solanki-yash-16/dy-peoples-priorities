# ⚙️ Backend API - DY Peoples Priorities

This is the Express REST API that powers the DY Peoples Priorities platform.

## Technologies
- Node.js & Express 5
- TypeScript
- MongoDB (Mongoose)
- JWT Authentication
- Nodemailer

## Getting Started

1. Create a `.env` file (refer to the root README for variables).
2. Run `pnpm dev` from the root monorepo, or `npm run dev` in this directory to start the server.
3. Access the API at `http://localhost:5000/api/v1`.

## Documentation
- **Swagger UI**: Navigate to `http://localhost:5000/api-docs` when the server is running to interact with the API endpoints.
