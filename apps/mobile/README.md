# 📱 Citizen Mobile App - DY Peoples Priorities

React Native mobile application for citizens to log and track civic complaints.

## Features
- Geospatial complaint logging
- Photo and voice attachments
- Track status of submitted issues
- Dark mode support

## Getting Started

1. Ensure you have the React Native environment set up (Android Studio/Xcode).
2. Configure your local backend IP in `src/api/axios.ts` (or point to production).
3. Run `pnpm run android` or `pnpm run ios` from this directory.
