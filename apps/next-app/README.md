# 🖥️ Admin Dashboard - DY Peoples Priorities

Next.js 16 (App Router) dashboard for city officials and administrators to manage civic issues.

## Features
- Real-time complaint tracking and filtering
- AI-generated summaries and urgency scoring
- Geospatial heatmap visualizations
- Custom UI components via ShadCN and Tailwind CSS

## Getting Started

1. Create a `.env` file with `NEXT_PUBLIC_API_URL` pointing to the backend API.
2. Run `pnpm dev` to start the development server.
3. Access the dashboard at `http://localhost:3000`.
