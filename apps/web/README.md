# 🌐 Citizen Web App - DY Peoples Priorities

Vite + React web application for citizens to engage with their municipality from any browser.

## Features
- Lightweight, fast web interface
- Submit complaints online with media attachments
- Track progress and updates

## Getting Started

1. Configure any necessary API environment variables.
2. Run `pnpm dev` to start the development server.
3. Access the web app at `http://localhost:5173`.
